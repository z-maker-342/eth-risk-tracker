Scoring Logic

DXY:
+1 if up >=0.30%
+2 if up >=0.60%
Nasdaq:
+1 if -0.5% to -1.0%
+2 if below -1.0%
ETF Flow:
+1 if latest outflow
+2 if latest + previous outflow
ETF 7-Day Trend:
+1 if negative
-1 if positive
Late Re-check:
+1 if worsened
-1 if improved
ETH Trend:
+1 if falling
-1 if rising
Funding:
+1 if high long leverage
-1 if low/neutral leverage

Final interpretation: 0-2 LOW, 3-4 MEDIUM, 5+ HIGH.
