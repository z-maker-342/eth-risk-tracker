ETH Night Dump Risk Tracker V4

Semi-auto ETH risk tracker for GitHub Pages. It can auto-load ETH spot price and ETHUSDT funding from Binance public APIs. You still manually enter DXY, Nasdaq futures, and ETH ETF flows.

Important: This tool is a risk filter, not financial advice. Use it to avoid bad nights and improve discipline, not to predict exact tops or bottoms.

Scoring Logic

DXY:
+1 if up ≥0.30%
+2 if up ≥0.60%
Nasdaq:
+1 if -0.5% to -1.0%
+2 if below -1.0%
ETF Flow:
+1 if today outflow
+2 if 2-day outflow
Late Re-check:
+1 if worsened
-1 if improved
ETH Trend:
+1 if falling
-1 if rising
Funding:
+1 if high long leverage
-1 if low/neutral leverage

Final interpretation: 0–2 LOW, 3–4 MEDIUM, 5+ HIGH. Funding mapping: above +0.03% = high long leverage, below -0.03% = low/neutral leverage, otherwise normal.
